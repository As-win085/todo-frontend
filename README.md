# To-Do List Application – Frontend

This project is the Frontend UI for the To-Do List Application built using React.

It provides a simple and clean user interface to manage tasks and communicates with the backend API using Axios.

## Tech Stack

React.js

Axios

Tailwind CSS 


## Features Implemented

✅ Add a new task

✅ View all tasks

✅ Mark tasks as completed

✅ Edit a task

✅ Delete a task

✅ Connect to backend using Axios

✅ Display loading spinner

✅ Display error messages
